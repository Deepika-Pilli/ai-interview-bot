# AI Interview Preparation Bot

AI-powered interview preparation platform built using React, Node.js, Express, LangChain, and Groq API.

## Features

- AI Interview Chatbot
- Mock Interview Mode
- Resume ATS Analyzer
- PDF Resume Upload
- Role-Based Interview Questions
- Resume-Based Interview Questions

## Tech Stack

Frontend:
- React
- Vite
- CSS

Backend:
- Node.js
- Express.js
- LangChain
- Groq API

Database:
- MongoDB (optional future scope)
